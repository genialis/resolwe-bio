# Docker image for common bioinformatics utilities

It is based on `ubuntu-18.04` version of [`docker.io/resolwebio/base`](
https://hub.docker.com/r/resolwebio/base/) image.

Included tools:
---------------
* bedGraphToBigWig (kent-v365)
* bedToBigBed (kent-v365)
* Bedtools (2.27.1)
* Bowtie (v1.2.2)
* Bowtie2 (v2.3.4.1)
* BWA (v0.7.17-r1188)
* Picard (2.8.1+dfsg-3)
* deepTools (3.1.0)
* FastQC (v0.11.8)
* HISAT2 (2.1.0)
* HTSeq (0.9.1)
* MultiQC (1.8.0b)
* QoRTs (1.3.0)
* Rsamtools (2.0.0)
* Samtools (1.9)
* Seqtk (1.2-r94)
* SRA Toolkit (v2.9.0)
* Subread (1.6.3)
* Tabix (1.7-2)
* WALT (1.01)
