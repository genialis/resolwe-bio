# Docker image for common bioinformatics utilities

It is based on `ubuntu-20.04` version of [`public.ecr.aws/genialis/resolwebio/base`](
https://hub.docker.com/r/resolwebio/base/) image.

Included tools:
---------------
* bedGraphToBigWig (kent-v365)
* bedToBigBed (kent-v365)
* Bedtools (2.30.0)
* Bowtie (v1.2.3)
* Bowtie2 (v2.3.5.1)
* BWA (v0.7.17-r1188) 
* bwa-mem2 (v2.2.1)
* Picard (2.18.25)
* deepTools (3.5.0)
* FastQC (v0.11.9)
* HISAT2 (2.2.1)
* HTSeq (0.13.5)
* MultiQC (1.9.0)
* QoRTs (1.3.0)
* Rsamtools (2.2.3)
* Samtools (1.11)
* Seqtk (1.3-r106)
* Subread (2.0.1)
* Tabix (1.10.2-3)
* WALT (1.01)
