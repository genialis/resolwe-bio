# Docker image for RNA-Seq processes

It is based on `ubuntu-17.10` version of [`docker.io/resolwebio/base`](
https://hub.docker.com/r/resolwebio/base/) image.

Included bioinformatics tools:
------------------------------
* affy (v1.56.0)
* BBMap (v37.90)
* Cufflinks (v2.2.1)
* Cutadapt (v1.15)
* DESeq2 (v1.16.1)
* FastQC (v0.11.5)
* HISAT2 (v2.1.0)
* HTSeq (v0.9.1)
* RSEM (v1.4.0)
* rtracklayer (v1.38.0)
* SAMtools (v1.5)
* Salmon (0.7.2)
* SRA Toolkit (v2.8.2-1)
* STAR (v2.5.3a)
* Trimmomatic (v0.36)
* tximport (v1.6.0)
